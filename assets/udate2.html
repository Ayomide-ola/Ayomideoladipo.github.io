After completing a Data Analytics for Business honors degree at CCT College and a specialized certificate program in Python Programming and Data Analytics from University College Dublin, I’ve been dedicated to mastering the tools and techniques that define modern data science.

I'm also adept with Microsoft Azure for managing large-scale data storage solutions and Excel for data manipulation. These tools enable me to extract meaningful insights, tell stories through data visualization, and build models that support data-driven decision-making.
